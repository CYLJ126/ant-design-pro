react ant design pro 项目中，我想封装一个表格：
样式：最上方为按钮行，其次为统计行，然后是表格，表格要有表头、数据行、下方分页栏；
按钮行：支持传入一个操作函数数组（包括函数中文按钮名，函数引用），渲染成一组按钮，放置于表格上方，可点击对应的操作函数按钮，执行相应的操作，入参可能没有，也有可能是选中的一行或多行数据；
统计行：从后端数据中获取，statistics字段，其为map结构，如{{总笔数，5},{总金额，500}}，需要格式化为“总笔数：5，总金额：500”，格式见下文；
表格：支持表头字段配置，支持向后端分页查询，使用ant design的pagenation组件支持；
后端查询方法为传入的一个函数参数，以及一个查询参数，后端返回格式：
{
	code: "100000",
	success:  true,
	currentPage: 5,
	totalPage: 10,
	data: [
	{每行记录},
	{每行记录}...
	],
	statistics:{
		totalNum: 5,
		totalAmount: 500.00
	}
}
当查询参数变化时，自动查询后端并刷新表格数据；
当点击分页参数时，自动组合查询参数和分页参数，向后端查询并刷新表格数据；
表头属性：顺序（从左到右，按顺序从小到大排序）、列名（用于显示）、列字段（用于从后端返回的行记录取对应的属性值）、各列宽度、是否支持排序、是否固定在左侧；
支持排序时，字段右侧显示上下箭头，支持升序或降序排序，排序时，传递相应参数请求后端数据，并刷新列表页；
每行第一列为当页行记录序号，第二列为复选框，支持勾选，保持固定在左侧；
表格数据支持单选和多选，以用作操作按钮参数；
